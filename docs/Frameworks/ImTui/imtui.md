# ImTui

> [Source Github](https://github.com/ggerganov/imtui?tab=readme-ov-file)

Đây là công cụ siêu mạnh mẽ để viết ứng dụng TUI
Đã kiểm chứng và vẽ ngon lành trên console, xịn vcl

Installing Lib

```text
sudo apt install libcurl4-openssl-dev
```

Getlib

```text
git clone https://github.com/ggerganov/imtui --recursive --depth=1
```

Install again

```text
git clone https://github.com/ggerganov/imtui --depth=1
git submodule update --init --recursive
```

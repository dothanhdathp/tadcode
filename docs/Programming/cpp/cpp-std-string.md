# \[C++\] String

## Khái niệm

- __String__ về khái niệm là một chuỗi các ký tự. Vốn là một chuỗi dữ liệu liên tiếp để biểu diễn một đoạn văn bản.
- __String__ được đóng gói trong thư viện gốc `std::string`
- __String__ về bản chất là một mảng các ký tự _(char array)_
- Thư viện `std::string` có nhiệm vụ đóng gói, mở rộng, chuẩn hóa các thao tác trên chuỗi sao cho an toàn.

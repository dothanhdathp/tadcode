# \[C++\] Install for Linux

- Tải về `g++`

```cpp
sudo apt-get update
sudo apt-get install g++
```

- Kiểm tra _binary_

```cpp
$ which c++
/usr/bin/g++
```

- Kiểm tra số hiệu phiên bản

```bash
$ g++ --version
g++ (Ubuntu 11.4.0-1ubuntu1~22.04) 11.4.0
```

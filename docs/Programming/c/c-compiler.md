# \[C\] Compiler

__Trình Biên Dịch__ này đơn giản nói về việc chuyển dịch dữ liệu từ __*C source file*__ sang thành mã máy.

Phần này không đi sâu vào cụ thể trình biên dịch mà chỉ đơn giản nhằm mục đích nghiên cứu quá trình biên dịch. Từng trình biên dịch khác nhau sẽ có áp dụng một số thay đổi khác nhau tuỳ thuộc vào mục đích sừ dụng và nghiên cứu.

Một số bộ công cụ biên dịch cho C gồm:

- [GNU GCC](/Tools/tools-gnu-gcc)
- [GNU G++](/Tools/tools-gnu-g++)

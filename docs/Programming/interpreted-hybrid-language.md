# Interpreted + Hybrid Language

## Interpreted Language
> Ngôn ngữ thông dịch

__Interpreted Language__ là dạng ngôn ngữ lập trình linh động, thường có tốc độ tải cao và đa nền tảng.

### 

## Hybrid Language

__Hybrid Language__ nghĩa là lai. Hầu hết các _ngôn ngữ thông dịch_ về sau đều dịch chuyển dần lên ngôn ngữ __*hybrid*__, tức là sẽ hỗ trợ cả biên dịch lẫn thông dịch. Dù sao thì ... thứ cuối cùng hoạt động vẫn là mã máy thuần túy.

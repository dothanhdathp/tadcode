# Rust Compound Types

Compound Types _(Các loại hợp chất)_ hay còn gọi là các loại phức tạp hơn các loại trước đó __*(Scalar Types)*__. Có ba loại phức tạp đó là:

- [Tuples](rust-tuples.md): Tập hợp cấu thành từ nhiều loại dữ liệu không cùng kiểu.
- [Array](rust-array.md): Tập hợp cấu thành từ nhiều loại cùng kiểu dữ liệu.
- [Slices](rust-slices.md): Tập hợp cấu thành từ nhiều loại cùng kiểu dữ liệu.

# \[KaTex\] Ký Tự Toán Học

## Số Học Cơ Bản

Các phép toán sẽ dụng hai chữ cái $A$ (`A`) và $B$ (`B`) làm ví dụ.

|      Text      |     KaTex      |      Text      |     KaTex      |
| :------------: | :------------: | :------------: | :------------: |
|     `A+B`      |     $A+B$      |     `A-B`      |     $A-B$      |
|     `A*B`      |     $A*B$      |     `A/B`      |     $A/B$      |
|  `A \times B`  |  $A \times B$  | `\frac{A}{B}`  | $\frac{A}{B}$  |
| `\cfrac{A}{B}` | $\cfrac{A}{B}$ | `\tfrac{A}{B}` | $\tfrac{A}{B}$ |
| `\dfrac{A}{B}` | $\dfrac{A}{B}$ |                |                |
|     `A^B`      |     $A^B$      |   `\bar{A}`    |   $\bar{A}$    |
|   `A^\circ`    |   $A^\circ$    |   `\sqrt{A}`   |   $\sqrt{A}$   |

## So Sánh

|        Text        |     KaTex     |     Text     |    KaTex     |
| :----------------: | :-----------: | :----------: | :----------: |
|       `A=B`        |     $A=B$     | `A \not = B` | $A \not = B$ |
|   `A \approx B`    | $A \approx B$ | `A \simeq B` | $A \simeq B$ |
| `A<B`or `A \lt B`  |     $A<B$     |  `A \leq B`  |  $A \leq B$  |
|    `A \leqq B`     |  $A \leqq B$  |  `A \lll B`  |  $A \lll B$  |
| `A>B` or `A \gt B` |     $A>B$     |  `A \geq B`  |  $A \geq B$  |
|    `A \geqq B`     |  $A \geqq B$  |  `A \ggg B`  |  $A \ggg B$  |

# \[KaTex\] Ký Tự Toán Học

## Số Học Cơ Bản

Các phép toán sẽ dụng hai chữ cái $A$ (`A`) và $B$ (`B`) làm ví dụ.

|    Text    |   KaTex    |               |               |                |                |                                                Note                                                 |
| :--------: | :--------: | :-----------: | :-----------: | :------------: | :------------: | :-------------------------------------------------------------------------------------------------: |
|   `A+B`    |   $A+B$    |               |               |                |                |                                                                                                     |
|   `A-B`    |   $A-B$    |               |               |                |                |                                                                                                     |
|   `A*B`    |   $A*B$    | `A \times B`  | $A \times B$  |                |                |                                                                                                     |
|   `A/B`    |   $A/B$    | `\frac{A}{B}` | $\frac{A}{B}$ | `\cfrac{A}{B}` | $\cfrac{A}{B}$ | Ngoài ra còn có `\dfrac`, `\tfrac`, `\genfrac`, `A \above B` và `{A \over B}` nhưng chúng như nhau. |
|   `A^B`    |   $A^B$    |               |               |                |                |                                                                                                     |
| `\bar{A}`  | $\bar{A}$  |               |               |                |                |                                                                                                     |
| `A^\circ`  | $A^\circ$  |               |               |                |                |                                                                                                     |
| `\sqrt{A}` | $\sqrt{A}$ |               |               |                |                |                                                                                                     |

## Creole

=== "Creole"
    <div style="display: flex; gap: 20px;">
      <div style="flex: 1;">
        ```text
        @startmindmap
        *:==Creole
        This is **bold**
        This is //italics//
        This is ""monospaced""
        This is --stricken-out--
        This is __underlined__
        This is ~~wave-underlined~~
        --test Unicode and icons--
        This is <U+221E> long
        This is a <&code> icon
        Use image : <img:https://plantuml.com/logo3.png>
        ;
        @endmindmap
        ```
      </div>
